# ⚠️ 本项目已停止维护

该仓库已不再进行主动维护，Issue 和 PR 可能不会得到及时响应，如有需要请自行 Fork。


## 这个项目是怎么来的

在做这个项目之前，我基本上都是跟着视频敲代码，有人带着走还好，但只要脱离了视频，让我自己从头搭一个项目，说实话还是挺难受的，感觉自己根本没真正学会。

所以就想着给自己一个机会，不看视频，不跟教程，自己去翻了一些优秀的开源项目和框架，然后独立把 Spring Security、Redis 这些技术整合在一起，硬是从零把这个项目做出来了。对我来说这算是一个比较重要的突破，至少证明自己离开视频也能独立完成一个项目。

不过毕竟是学习阶段做的东西，里面还有不少 Bug 没处理，加上临近毕业要找工作，实在没精力继续维护了，就先放在这里吧。


## 后续计划

最近在重新开发一个新项目，不管是模块设计还是代码质量都比这个要好不少，算是一次比较彻底的重构。

等新项目做完之后，我会把里面跟业务无关的基础框架部分单独抽出来，去掉业务逻辑，整理成一个干净的基础模板，到时候用来替换这个项目。有进展的话会在这里更新链接。
